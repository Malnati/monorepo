# Changelog - Correção de Sintaxe YAML no Workflow Prettier

**Data/Hora UTC:** 2025-11-18 16:52:28

## Arquivos Modificados

### Modificados
- `.github/workflows/prettier.yml` - Removida permissão inválida `workflows: read` que causava erro de sintaxe YAML

## Problema Resolvido

Após submissão do workflow para o GitHub, foi identificado erro de sintaxe YAML:
```
Invalid workflow file: .github/workflows/prettier.yml#L1
(Line: 14, Col: 7): Unexpected value 'workflows'
```

O problema estava na linha 14, onde havia uma permissão inválida `workflows: read`. Esta não é uma permissão válida no GitHub Actions.

## Solução Implementada

Removida a linha `workflows: read` da seção `permissions` do workflow. A permissão `contents: write` é suficiente para o workflow do Prettier fazer commits das alterações de formatação.

## Validações Realizadas

- ✅ Validação de sintaxe YAML: Workflow validado com sucesso usando parser Python
- ✅ Estrutura de permissões: Apenas permissões válidas do GitHub Actions mantidas
