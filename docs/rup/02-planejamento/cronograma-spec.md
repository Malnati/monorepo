# Cronograma Macro

> Base: [./cronograma.md](./cronograma.md)


| Fase | Período | Entregas Principais |
| --- | --- | --- |
| Iniciação | Mês 1-2 | Visão do produto, mapa de stakeholders, definição de indicadores de impacto. |
| Elaboração | Mês 3-4 | Arquitetura detalhada, protótipo de onboarding, acordos com cooperativas piloto. |
| Construção | Mês 5-8 | Desenvolvimento dos microsserviços core, apps web/mobile e pipelines de dados. |
| Transição | Mês 9-10 | Testes integrados, certificação regulatória, piloto com 3 cidades. |
| Implantação | Mês 11-12 | Lançamento público controlado, monitoramento intensivo e ajustes de governança. |

## Marcos Dependentes
- Aprovação do BACEN para operar como Sociedade de Crédito Direto até o final do mês 4.
- Contratos de fornecimento de resíduos com pelo menos cinco cooperativas regionais até o mês 6.
- Integração com registradora de créditos de carbono homologada antes do piloto público.
