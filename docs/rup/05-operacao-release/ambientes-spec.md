# Ambientes (Histórico)

> Base: [./ambientes.md](./ambientes.md)
> Referência atualizada: [Ambientes e Configurações](../05-entrega-e-implantacao/ambientes-e-configuracoes-spec.md)

Este arquivo preserva o vínculo histórico da fase Operação e Release com o catálogo de ambientes do App. Utilize-o apenas para rastrear decisões antigas; toda configuração vigente está descrita em [Ambientes e Configurações](../05-entrega-e-implantacao/ambientes-e-configuracoes-spec.md), que atende [REQ-101](../02-planejamento/requisitos-spec.md#req-101), [REQ-201](../02-planejamento/requisitos-spec.md#req-201) e mantém equivalência com os legados `REQ-001`, `REQ-014`.

Sempre que atualizar domínios ou variáveis, certifique-se de apontar o changelog correspondente e referenciar os requisitos afetados. Mudanças que envolvam colaboração humana devem registrar a equivalência com `REQ-031`–`REQ-033` (legados) e com [REQ-304](../02-planejamento/requisitos-spec.md#req-304).
