# Publicação (Histórico)

> Base: [./publicacao-web-store.md](./publicacao-web-store.md)
> Referência atualizada: [Publicação e Versionamento](../05-entrega-e-implantacao/publicacao-e-versionamento-spec.md)

Os fluxos atuais de publicação do App são controlados pela fase [Entrega e Implantação](../05-entrega-e-implantacao/README-spec.md). Esta página preserva o legado da distribuição via Chrome Web Store, garantindo rastreabilidade com [REQ-019](../02-planejamento/requisitos-spec.md#req-019), [REQ-022](../02-planejamento/requisitos-spec.md#req-022), [REQ-029](../02-planejamento/requisitos-spec.md#req-029) e, historicamente, com `REQ-018`, `REQ-022`, `REQ-029`, `REQ-030`.

Se precisar reconstituir o processo antigo (extensão), utilize o changelog do período correspondente e assegure que quaisquer retomadas estejam alinhadas à governança atual e às notas de colaboração humana (`REQ-031`–`REQ-033`).
