# APP Brand Assets Directory

Este diretório contém os ativos visuais neutros e tokens mínimos de identidade para o template APP.

## Logos autorizadas (SVG)

- `app-logo.svg`: logotipo vetorial "APP" com gradiente verde neutro. Use em fundos claros ou escuros, mantendo respiro mínimo de 16px.

## Tokens e placeholders

- **Nome padrão:** APP (substituível via variáveis de ambiente ou textos de UI).
- **Paleta base:** gradiente `#0F766E` → `#22C55E`, fundo claro `#ECFDF3`.
- **Canais:** e-mail de contato e domínio devem ser preenchidos com valores do projeto (ex.: `contato@example.com`, `app.localhost`).

## Diretrizes de uso

- ✅ Manter o SVG sem textos adicionais ou slogans.
- ✅ Redimensionar proporcionalmente, preservando contraste WCAG AA.
- ❌ Não inserir nomes de empresas legadas ou domínios anteriores nos arquivos versionados.
