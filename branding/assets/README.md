# APP Brand Assets Directory

Este diretório contém os ativos visuais oficiais da marca APP autorizados para uso no projeto.

## Logos Autorizadas

**IMPORTANTE:** Apenas as seguintes logos estão autorizadas para uso:

### 1. Logo com Transparência - Versão Colorida (`dominio-logo-transparencia-colors.png`)
- **Formato:** PNG com transparência, otimizado (42KB)
- **Uso recomendado:** Fundos claros, neutros ou coloridos
- **Contextos ideais:**
  - Fundos brancos ou cores muito claras (`--color-neutral-05` a `--color-neutral-30`)
  - Superfícies limpas sem texturas ou padrões
  - Sobreposição em imagens e fotografias com fundos claros
  - Materiais impressos em alta qualidade
  - Apresentações formais e documentos oficiais
  - Páginas web com fundo claro, branco ou colorido
  - Favicons, avatares e ícones de navegação em contextos claros
- **Contraste:** Garante contraste WCAG AA em fundos claros
- **Área de respiro:** Mínimo de 16px em todos os lados

### 2. Logo com Transparência - Versão Preto e Branco (`dominio-logo-transparencia-bw.png`)
- **Formato:** PNG com transparência, otimizado (33KB)
- **Uso recomendado:** Fundos escuros ou contextos monocromáticos
- **Contextos ideais:**
  - Fundos escuros ou com gradientes
  - Interfaces com temas escuros (dark mode)
  - Aplicações web sobre backgrounds variados escuros
  - Hubs onde o espaço é restrito ou contexto visual é dinâmico
  - Contextos que exigem alto contraste monocromático
  - Impressões em preto e branco
  - Documentos técnicos e relatórios formais
- **Contraste:** Otimizada para manter legibilidade em fundos diversos
- **Área de respiro:** Mínimo de 16px em todos os lados

## Diretrizes de Uso

### ✅ Permitido
- Utilizar apenas as duas logos autorizadas acima
- Redimensionar proporcionalmente mantendo aspect ratio
- Aplicar sobre fundos que garantam contraste WCAG AA
- Usar em materiais oficiais do projeto APP

### ❌ Proibido
- Modificar as cores, formas ou elementos das logos
- Adicionar efeitos, sombras ou distorções
- Utilizar logos com textos adicionais, slogans ou microtextos
- Comprimir ou distorcer as proporções
- Aplicar sobre fundos que prejudiquem a legibilidade
- Usar versões não autorizadas das logos

## Terminologia Oficial

- **Termo correto:** "APP" ou "plataforma APP"
- **Contextos:** Toda interface do usuário, formulários, políticas LGPD, termos de uso
- **Nota:** O termo "App" está reservado para futura fase Fintech/Moeda
